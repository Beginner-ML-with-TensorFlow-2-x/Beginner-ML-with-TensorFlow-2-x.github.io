# Beginner-ML-with-TensorFlow-2-x.github.io
Organisation GitHub Page

## Basic Usage

### Download manually

```sh
git clone https://github.com/https://github.com/Beginner-ML-with-TensorFlow-2-x/ml4rookies-site
cd ml4rookies-site
```

### Setup

```sh
npm i
```

### Run

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Serve built app

```sh
npm run serve
```
